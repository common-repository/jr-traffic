== Description ==

This plugin allows you to display the latest traffic updates directly on your blog - From any zip code you wish!

Simply enable the plugin through your admin panel, edit the JR Traffic settings in the Settings --> JR Traffic menu and enable the widget through the Appearance --> Widgets menu! Traffic updates will now be displayed on your blog!

== Installation ==

1. Upload `jr-traffic.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Edit the JR Traffic Settings in your Administration Panel
4. Enable the Widget through the Appearance --> Widgets menu.
